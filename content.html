In HTML, the <pre> tag is used to display preformatted text. Unlike normal paragraph text, whitespace inside a <pre> element is preserved, including spaces, tabs, and line breaks.

For example, the following content is displayed with the same spacing and line breaks as written:

    This is a sample of preformatted text.
        It preserves spaces and line breaks.

This was created using the following HTML code:

<pre>
    <strong>This is a sample of preformatted text.</strong>
        It preserves spaces and line breaks.
</pre>

HTML is still parsed inside pre. In the example above, the <strong> tag is interpreted as HTML, so the text "This is a sample of preformatted text." is displayed in bold.
